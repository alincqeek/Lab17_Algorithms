## Что такое алгоритм?

**Алгоритм** — это порядок действий для решения задачи.

### Пример из жизни:
Как приготовить чай:
1. Взять чайник
2. Налить воду
3. Вскипятить
4. Залить в чашку
5. Добавить заварку

---

## Блок-схемы

**Блок-схема** — это рисунок алгоритма.

### Фигуры в блок-схемах:

| Фигура | Название | Что значит |
|--------|----------|------------|
| Овал | Начало / Конец | Старт или финиш |
| Прямоугольник | Действие | Что делаем |
| Ромб | Условие | Вопрос (да/нет) |
| Параллелограмм | Ввод / Вывод | Вводим или выводим |
| Стрелка | Связь | Куда идти дальше |
